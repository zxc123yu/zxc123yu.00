0x02 正则表达式模块的运用

  1. .match(pattern, string, flags)方法
    判断是否匹配，如果匹配成功，返回一个Match对象，否则返回None
  2. .split()方法
    用正则表达式切分字符串
  3. .compile()方法
    用于生成一个独立的规则模式，如：regex=re.compile(r'\w*o\w*')
  4. .sub(pattern，repl，string，count，flags)方法
    用于通过正则式来实现字符串的替换处理
    • pattern：表示正则中的模式字符串
    • epl，表示要被替换的，可以是字符串也可以是函数，
      • 如果是字符串，则所有的反斜杠转义字符都会被处理
      • \n：被处理为对应的换行符
      • \r ：被处理为回车符
      • 不能被识别的转义字符，则只是被识别为普通的字符，例如：\j被处理为j这个字母本身
      • 反斜杠加g以及中括号内一个名字，即\g<name>对应命名了的组
    • string：要处理的字符串
    • count：限定替换的个数，默认为替换所有
    • flags：匹配模式，可以使用按位或‘|‘表示同时生效，也可以在正则表达式中指定。
      • re.I 忽略大小写
      • re.L 表示特殊字符集\w,\W,\b,\B,\s,\S
      • re.M 表示多行模式
      • re.S ‘.’包括换行符在内的任意字符
      • re.U 表示特殊字符集\w,\W,\b,\B,\d,\D,\s,\D

0x03 与mysql的联合运用-操作数据库

  1. 使用pymysql.connect(host,post,username,password,database)来连接数据库，database参数可为空，返回值为可操作数据库的句柄。
  2. 可对句柄调用.cursor()方法获取操作指令光标用于数据库操作。若连接时所选数据库为空，则应对光标调用.execute()方法执行数据库使用指令。
  3. 对于光标在每个过程结束后都应立即调用.close()方法进行销毁，对于句柄在完成所有的数据库操作后也应立即调用.close()方法进行销毁。
  4. .commit()方法用于执行插入、更新、删除指令后提交数据库执行。
  5. 对于查询指令，在光标执行命令结束后，可对光标调用以下三种方法：
    • fetchone()：该方法获取下一个查询结果集。结果集是一个对象
    • fetchall()：接收全部的返回结果行.
    • rowcount：这是一个只读属性，并返回执行execute()方法后影响的行数。
  6. 在出现错误时，可对光标调用.rollback()方法来回滚当前游标的所有操作

0x04 BeautifulSoup使用

  1. 解析方式

  2. findAll() 和 find() 方法

    • 定义：

    • 参数声明：

      1. tag：标签参数，可定义一个标签名称或由多个标签名称组成的列表，类似于一个“或”关系的过滤器
      2. attributes：属性参数，用一个字典封装一个标签的若干属性和对应的属性值
      3. recursive：递归参数，是一个布尔变量，为True时,会根据要求去查找标签参数的所有子标签，以及子标签的子标签，默认值为True
      4. text：文本参数，使用标签的文本内容去匹配
      5. limit：范围限制参数，只能使用与findAll方法中
      6. keyword：关键词参数，可以选择具有指定属性的标签
    • 区别：
      • findAll()方法返回的是符合要求的所有元素的列表，而find()方法则是直接返回符合要求的结果
      • find()方法等同于findAll()方法中的limit参数设置为1时的特例
